# Screenshots

Captured via Playwright against a local `npm run preview` build. Two viewports
per screen:

- **`*-iphone-14.png`** — iPhone 14 (390 × 844), the primary mobile target
- **`*-ipad-air.png`** — iPad Air (810 × 1080), responsive check

Screens:

| File | What |
|------|------|
| `home-*` | Teams + Characters grid, Alpha/Epsilon/Omega selector, "All events" entry to Atlas |
| `team-xmen-*` | X-Men (Complete Modern Era) team detail — chronological event cards |
| `event-hom-*` | House of M event — issue list with role badges and Marvel cover thumbs |
| `event-axis-*` | A.X.E.: Judgment Day — example of a non-X-Men crossover with X-Men tie-ins |
| `character-cyclops-*` | Hand-curated Cyclops spine with notes ("Debut of Scott Summers", "Dark Phoenix Saga climax") |
| `atlas-*` | Transit-map Atlas — teams as colored lines, events as stations, interchanges where lines meet |
